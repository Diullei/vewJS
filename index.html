vewJS

Video Element Watcher - A script to watch <video> element

This uses the W3C code used to show all properties and events that the tag <video> supplies to the browser.

You can check the W3C HTML5 Video Events and API.

Still testing. To do it, you have to:

  • Add a class named vew to the <video> element
  • Access the vewJS demo page
  • Add the vewJS button to your bookmark (or click it to try before)
  • Play the video and watch things happening :)

All the working versions will be at the master branch.

All work in progress will be at dev branch.
